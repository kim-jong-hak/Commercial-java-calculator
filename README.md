# 맥북용 상용 java 계산기 프로젝트 

### 기능 구현
- 연산자 우선순위 
- 부동소수점 해결을 위해 "BigDecimal" 사용
- 히스토리(계산 기록) 
- 지뢰찾기(게임)
- "+" , "-" , "*" , "/" , "=" 연산기능
- 계산작업 후 자릿수 표현 (ex 1,000)
